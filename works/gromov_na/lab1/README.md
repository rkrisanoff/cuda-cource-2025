# Lab 1: CUDA Graphics - Вывод геометрических фигур в терминал

## Описание

Программа выводит квадрат, используя параллельные вычисления на CUDA. Каждый поток обрабатывает
один "пиксель" изображения для отрисовки границы квадрата.

## Требования

- CUDA Toolkit 11.0 или выше
- GPU с compute capability 7.0+
- Linux с GCC 9.0+
- Make

## Компиляция

```bash
make
```

## Запуск

```bash
./square [width] [height]

# Примеры:
./square 15 15
./square 20 10
```

## Результаты

```
Размер: 15x15
Фигура: квадрат

***************
*             *
*             *
*             *
*             *
*             *
*             *
*             *
*             *
*             *
*             *
*             *
*             *
*             *
***************
```

## Особенности

- Используется 2D сетка потоков для параллельной обработки
- Каждый поток рассчитывает одну позицию
- Линеаризация 2D сетки потоков в 1D индекс
- Проверка границ для предотвращения выхода за пределы

## Замечания

- Максимальный размер ограничен количеством потоков на блок
- Для больших размеров может потребоваться использование нескольких блоков
- Используется CUDA_CHECK макрос для обработки ошибок
