# Lab 1: CUDA Graphics

## Описание

Цель лабораторной работы — реализовать параллельное вычисление принадлежности точек окружности с помощью технологии CUDA и отобразить результат в текстовом терминале.  

Каждая точка на сетке 80×24 проверяется независимым потоком GPU: если точка находится внутри или на границе круга заданного радиуса, в соответствующей позиции выводится символ `*`, иначе — пробел.

## Компиляция

```bash
make
```

## Запуск 

```bash
./circle
```

## Результаты

* Программа корректно отображает круг в центре терминала размером 80×24.
* Все вычисления выполняются на GPU, вывод — на CPU.
* Тестирование проведено на GPU с архитектурой sm_70 и выше.

## Замечания

* Из-за не квадратной формы символов в терминале круг выглядит слегка сжатым по вертикали. Для более точного отображения можно скорректировать формулу (например, умножить `dy*dy` на `2`).
* Программа использует явную синхронизацию (cudaDeviceSynchronize()) для гарантии завершения вычислений перед выводом.
