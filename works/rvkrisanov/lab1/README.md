# Lab 1: CUDA Graphics - Вывод квадрата

## Описание
Программа рисует квадрат в терминале используя CUDA kernel. Каждый символ вычисляется параллельно на GPU, после чего результат копируется на хост и выводится в консоль.

## Компиляция
```bash
make
```

## Запуск
```bash
./square
```
